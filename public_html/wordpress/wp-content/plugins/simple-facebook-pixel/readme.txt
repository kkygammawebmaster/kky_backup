== Description ==

Adds the ability to add a Facebook tracking pixel to any page.

== Installation ==

1. Upload `facebook_pixel.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Open up the Facebook Pixel settings page
4. Select the page you want the pixel displayed on, enter the code to the paragraph box
5. Save the settings
